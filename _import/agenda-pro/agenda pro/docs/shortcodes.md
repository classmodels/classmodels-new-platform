# Class Models Agenda Pro v3 shortcodes

## Frontend boeken
- `[portfolio_agenda_afspraak]` — portfolio afspraak boeken.
- `[opleiding_inschrijven]` — opleiding afspraak boeken.
- `[casting_afspraak]` — casting afspraak boeken.
- `[gratis_fotoshoot_afspraak]` — gratis fotoshoot afspraak boeken.
- `[intake_gesprek_afspraak]` — intake gesprek boeken.
- `[cm_agenda_booking calendar="casting"]` — boekingsformulier voor eender welke agenda-slug.

## Kalender en lijsten
- `[cm_agenda_calendar]` — kalender/lijstoverzicht van afspraken.
- `[cm_agenda_mijn_afspraken]` — afspraken van ingelogde gebruiker.

## Admin/frontbeheer
- `[cm_agenda_slots_admin calendar="portfolio"]` — momentenlijst per agenda.
- `[cm_agenda_bookings_admin calendar="casting"]` — afsprakenlijst per agenda.
- `[cm_agenda_shortcode_lijst]` — deze lijst in de site tonen.

## Standaard agenda-slugs
- `portfolio`
- `opleiding`
- `casting`
- `gratis-fotoshoot`
- `intake-gesprek`
